Introduction

Conventions of this Guide

Within this guide, text of particular importance is styled so it can be differentiated from normal text. Given that this guide focuses on a programming course, the primary piece of textual information most students will be looking for is code. Code within this guide occurs two forms: firstly, short code segments are displayed on a single line, set in monospace and with a dotted underline. The code statement print("Hello world!") should be visually distinct from the rest of this block. On the other hand, long-form segments of code, commonly called "blocks" of code, are shown in a box with a light gray background, set in monospace. Each code block also contains line numbers on the left side of the box, which start at 1 in every new block. There will never be two boxes representing different sections of the same block; each box will always contain the entirety of one block. As blocks of code in their entirety are likely to become the primary reason for using this guide, they should be impossible to miss; see below for an example of a code block. 

#!/usr/bin/env python3

def say_hello(name):
	print(f"Hello {name}!")

if __name__ == "__main__":
	say_hello("Alice")

The usage of keyboard shortcuts within the course is essentially mandated. Keyboard shortcuts are shown in a small box with a light gray background, set in monospace. The keyboard shortcut Ctrl+S represents pressing the Ctrl and S keys at the same time. Keyboard keys will always be shown in uppercase regardless of if the Shift key is pressed. Common keyboard modifiers, such as the control key, the Windows key, the alt key, and the function key are abbreviated Ctrl, Win, Alt, and Fn respectively. Note than on some computers, the function key inverts the functionality of the top row of keys, rather than activating the secondary keys. If this is the case with the computer you are using, ignore the function key unless it is required to use another keyboard shortcut.

The last text styling convention you should be aware of is the styling of vocabulary and other notable text. Notable text is italicized, bolded, and has a dotted underline. The vocabulary word algorithm should show that this styling makes notable text stand out from the rest of this paragraph. In general, all text that is considered notable is styled this way, and all text that is styled this way can be entered into the search box.
